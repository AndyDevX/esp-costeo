# Guía comandos Git

Ctrl + Ñ
    Abrir la terminal de comandos

git status
    Comprobar el estado actual del repositorio

git add .
    Añadir todos los archivos modificados al registro

git commit -a -m "mensaje"
    Confirmar la modificación de los archivos en el registro

git commit -a --allow-empty-message -m ""
    Confirmar la modificación de los archivos en el registro, sin ningún mensaje

git push
    Enviar los cambios al repositorio principal

git pull
    Bajar los archivos nuevos y modificados desde el repositorio
